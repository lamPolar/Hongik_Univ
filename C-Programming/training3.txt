문제 description

bank database 만들기

은행의 운영에 필요한 작은 데이터베이스를 다음과 같이 만든다.

1)	두 배열, 즉 구좌번호 (accountID)를 저장하는 accountID[20] 와 잔고를 나타내는 balance [20]를 생성한다. 둘 다 int type일 것.
accountID 의 값 생성시에는 rand( )를 사용할 것. 사용시 #include <stdlib.h> 를 포함 시켜야 함.

2)	구좌번호는 모두 6자리 숫자이며, 맨 앞자리 숫자는 1이나 2이다.
1은 보통예금구좌를 의미하고, 2는 정기예금 구좌를 나타낸다.
*** 주의사항 : account번호 생성시 같은 번호가 이미 존재하는지 체크해야함.

3)	= 를 이용하여 처음 5개의 구좌와 잔고를 다음과 같이 채운다.
accountID[0] = 112036;   balance[0] = 300000;
같은 index를 가진 account와 balance는 항상 같이 붙어 다닌다.
즉, 위의 예를 보면 accounted[0]의 구좌번호 112036 의 잔고는 300000이다.
만약 어떤 이유로 112036이 accountID[5]로 이동 한다면 그 잔고인 300000도 balance[5]로 이동되어야 한다.

4)	Scanf를 이용하여 3개의 구좌와 잔고를, 즉 accountID[5] ~ account[7]를 더 생성하라.

5)	다음과 같은 기능을 추가하라.
  a)	지금까지 만든 구좌 중 잔고가 제일 많은 구좌를 찾아내는 기능.
  b)	지금까지 만든 구좌 중 잔고가 제일 적은 구좌를 찾아내는 기능.
  c)	잔고의 평균을 구하는 기능.
  d)	accountID값을 입력 받아 그 구좌의 잔고를 출력하는 기능.
      예를 들어 112036이 입력되면 거기에 맞는 잔고인 300000을 출력하라.
      만약 입력된 accountID 값이 없으면 account가 없다고 출력하라.
  e)	모든 구좌를 보이기
  f)	프로그램을 끝내기
